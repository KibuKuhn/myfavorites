Manual

  • Supported file types
    - All 'ordinary' files
    - Desktop files that hold a hyperlink
    - Links from a web browser. Drag & drop a link from the input field of the browser into myFavorites
    - Folders
    - Executable files are not supported

  • Position favoites display
    - When you start myFavorites the first time, favorites are being displayed at the center of the screen
    - Click into the display and drag it to the desired location
    - Open settings and unselect 'Adjust display position of favorites'
    - To change the location again, select 'Adjust display position of favorites' and proceed as before

  • Manage Favorites
    - Open myFavorites > Configure Favorites
    - Follow instructions
    - Use Ctrl Up and Ctrl Down respectively to move a favorite
    - Use Del Key to remove a favorite
    - Use Ctrl A to add an alias to the item. If present, it is displayed instead of the item's path
    - Use Ctrl B to create a box. Assort your favorites with boxes
    - Use Ctrl X, Ctrl C, Ctrl V to move and to copy favorites
    - Close window to apply changes

  • Maintain Settings
    - Right click onto myFavorites tray icon or
    - Open myFavorites > Settings
    - Change language and Look and Feel as required
    - Export favorites as JSON
    - Import favorites from JSON
    - Close window to apply changes

  • Help
    - Open myFavorites > Help

  • Exit myFavorites
    - Open myFavorites > Exit

  • Troubleshooting
    - If myFavorites cannot read the preferences, restart it with argument -Dclean. This will delete all preferences
    - If the tray icon looks like an unexpected error occured. Please restart myFavorites.

  • Logging
    - By default myFavorites.log is generated in the home diretory of the current user. The file is hidden.
    - To change the location, use -DlogDir=<your_log_folder>
    - To change the log level, use -DlogLevel=<new log level>
    when starting myFavorites.
