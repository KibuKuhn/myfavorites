Anleitung

  • Unterstützte Dateitypen
    - Alle 'normalen' Dateien
    - Desktop-Dateien die einen Hyperlink enthalten
    - Verzeichnisse
    - Ausführbare Dateien werden nicht unterstützt

  • Favoriten verwalten
    - myfavorites öffnen > Favoriten konfigurieren
    - Anweisungen folgen
    - Benutze Strg Auf bzw. Strg Ab um Favoriten zu verschieben
    - Benutze Entf-Taste um einen Favoriten zu entfernen
    - Benutze A-Taste um ein Alias für einen Favoriten zu setzen. Falls gesetzt, wird der Alias angezeigt anstatt des Pfads
    - Fenster schließen um Änderungen anzuwenden

  • Einstellungen verwalten
    - myfavorites öffnen > Einstellungen
    - Sprache bzw. Look and Feel nach Bedarf ändern
    - Fenster schließen um Änderungen anzuwenden

  • Hilfe
    - myfavorites öffnen > Hilfe

  • myfavorites beenden
    - myfavorites öffnen > Beenden

  • Probleme
    - Wenn myfavorites die Einstellungen nicht einlesen kann, mit Argument -Dclean neu starten. Alle Einstellungen werden damit gelöscht.
    - Wenn das Tray Icon so aussieht, ist ein unerwarteter Fehler aufgetreten. Bitte myfavorites neu starten.

  • Logging
    - myfavorites.log wird normalerweise im Home Verzeichnis des aktuellen Benutzers erzeugt. Die Datei ist versteckt.
    - Um das Verzeichnis zu ändern, benutze -DlogDir=<dein_log_Verzeichnis>
    - Um den Log Level zu ändern, benutze -DlogLevel=<neuer Log Level>
    beim Starten von myfavorites.
